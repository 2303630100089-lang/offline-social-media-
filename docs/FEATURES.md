## Performance Metrics

### Messaging Performance
- Message delivery time: < 100ms (nearby)
- Encryption overhead: < 5ms
- Database sync: < 500ms
- Search performance: < 1s (10k messages)

### Network Performance
- Bluetooth range: 10m (Classic), 50m (BLE)
- Wi-Fi Direct: 100m
- Throughput: 1-10 Mbps
- Latency: 50-200ms

### Battery Performance
- Idle consumption: < 5% per hour
- Active use: 20-30% per hour
- Standby: < 1% per hour
- Mesh scanning: 2-3% per hour

### Storage
- App size: ~80-120 MB
- Database size: 1 MB per 1000 messages
- Cache size: Configurable (default 100 MB)
- Offline maps: ~50-200 MB per region
